# IAW301 - Web Application Security
## Spring 2026 Course Information

### Course Overview

**Course Code**: IAW301  
**Course Name**: Web Security (An ninh Web)  
**Credits**: 3  
**Degree Level**: Bachelor  
**Prerequisites**: ITE303 or ITE302c or IAP201

### Time Allocation
- **Total Study Hours**: 150 hours
- **Contact Hours**: 45 hours (60 sessions × 45 minutes)
- **Final Exam**: 1 hour
- **Self-Study**: 104 hours

### Course Description

This course provides an immersive learning experience in the field of web application security, featuring the use of **PortSwigger** as our primary lab material. PortSwigger offers a powerful platform for hands-on exploration of web application vulnerabilities and security best practices. Through this course, students will gain a deep understanding of web application security principles and develop practical skills in a controlled, real-world environment.

### Primary Tools & Platforms

- **PortSwigger Web Security Academy** - Main learning platform
- **Burp Suite** - Professional web security testing tool
- **SQLMap** - Automated SQL injection testing tool
- **DVWA v1.0.7** - Damn Vulnerable Web Application for practice

### Course Learning Outcomes (CLOs)

Students who successfully complete this course will be able to:

1. **CLO1**: Gain an introductory understanding of web security principles and their importance in the context of web applications
2. **CLO2**: Understand the concept of information disclosure vulnerabilities and their potential impact on web application security
3. **CLO3**: Examine vulnerabilities in various authentication mechanisms (password-based and multi-factor authentication)
4. **CLO4**: Learn how to identify and exploit IDOR and Privilege escalation
5. **CLO5**: Explore vulnerabilities related to OAuth 2.0 authentication mechanisms
6. **CLO6**: Understand SQL injection vulnerabilities and their exploitation
7. **CLO7**: Understand OS Command Injection vulnerabilities and their exploitation
8. **CLO8**: Learn to identify and exploit reflected XSS vulnerabilities
9. **CLO9**: Understand SSRF vulnerabilities and the potential risks they pose
10. **CLO10**: Develop the skills to identify and exploit race conditions, especially related to limit overruns
11. **CLO11**: Understand the concept of path traversal vulnerabilities and how they can be exploited
12. **CLO12**: Gain the skills to identify and exploit vulnerabilities related to file uploads

### Course Structure (11 Chapters, 19 Labs)

#### Chapter 1: Web Application Fundamentals (Sessions 1-6)
- How web applications work
- Client-side vs Server-side security
- Security tools and scanners
- OWASP introduction
- **Lab 1**: Exploring the OWASP

#### Chapter 2: Information Disclosure (Sessions 7-9)
- Information disclosure vulnerabilities
- Impact assessment
- **Lab 2**: Information Disclosure Exploitation

#### Chapter 3: Authentication Vulnerabilities (Sessions 10-18)
- Password-based authentication flaws
- Multi-factor authentication bypass
- Other authentication mechanisms
- **Lab 3**: Password-based Authentication
- **Lab 4**: Multi-factor Authentication
- **Lab 5**: Other Authentication Mechanisms

#### Chapter 4: Access Control Vulnerabilities (Sessions 19-27)
- Insecure Direct Object References (IDOR)
- Privilege escalation
- OAuth 2.0 vulnerabilities
- **Lab 6**: IDOR Exploitation
- **Lab 7**: Privilege Escalation
- **Lab 8**: OAuth 2.0 Bypass

#### Chapter 5: SQL Injection (Sessions 28-36)
- UNION-based SQL injection
- Time-based blind SQL injection
- Error-based blind SQL injection
- **Lab 9**: SQL Injection UNION Attacks
- **Lab 10**: Time-based Blind SQL Injection
- **Lab 11**: Error-based Blind SQL Injection

#### Chapter 6: OS Command Injection (Sessions 37-42)
- Command injection vulnerabilities
- Blind command injection
- **Lab 12**: OS Command Injection
- **Lab 13**: Blind OS Command Injection

#### Chapter 7: Cross-Site Scripting (Sessions 43-48)
- Reflected XSS
- Stored XSS
- **Lab 14**: Reflected XSS
- **Lab 15**: Stored XSS

#### Chapter 8: Server-Side Request Forgery (Sessions 49-51)
- SSRF vulnerabilities and exploitation
- **Lab 16**: SSRF Exploitation

#### Chapter 9: Race Conditions (Sessions 52-54)
- Race condition vulnerabilities
- Limit overrun exploitation
- **Lab 17**: Race Conditions

#### Chapter 10: Path Traversal (Sessions 55-57)
- Directory traversal attacks
- **Lab 18**: Path Traversal

#### Chapter 11: File Upload Vulnerabilities (Sessions 58-60)
- File upload security flaws
- **Lab 19**: File Upload Exploitation
