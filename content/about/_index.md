# About This Blog

## Personal Learning Journey

This blog documents my personal learning journey in Web Application Security. All content here represents my own notes, experiences, and understanding gained through self-study and practice.

## What You'll Find Here

- **Personal lab notes** and solutions
- **My understanding** of various security concepts
- **Hands-on experiences** with security tools
- **Learning resources** I found helpful

## Topics Covered

Through 19 lab exercises, I explored:

1. **OWASP Top 10** - Understanding critical web security risks
2. **Authentication Security** - Password-based and multi-factor authentication
3. **Access Control** - IDOR, privilege escalation, OAuth vulnerabilities
4. **Injection Attacks** - SQL injection and OS command injection
5. **Cross-Site Scripting** - Reflected and stored XSS
6. **Server-Side Vulnerabilities** - SSRF, race conditions, path traversal
7. **File Upload Security** - Secure file handling practices
