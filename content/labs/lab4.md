# Lab 4: Authentication Vulnerabilities (multi-factor authentication)

## Objectives
- Understand MFA vulnerabilities
- Practice MFA bypass techniques
- Configure secure MFA

## Lab Content

### 1. Types of MFA Vulnerabilities
- **2FA bypass**: Bypassing two-factor authentication
- **TOTP vulnerabilities**: Time-based OTP vulnerabilities
- **SMS-based 2FA flaws**: SMS-based 2FA vulnerabilities
- **Backup codes abuse**: Backup code abuse

### 2. Attack Techniques
- Response manipulation
- Status code manipulation
- Session handling flaws
- Race conditions

### 3. Practice
- Bypass 2FA mechanisms
- Exploit backup codes
- Session fixation attacks
