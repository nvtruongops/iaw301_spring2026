# Web Security Lab Notes

This page contains all 19 lab exercises I've completed while learning Web Application Security. Each lab focuses on specific vulnerabilities and attack techniques.

## What I Learned

Through these labs, I gained understanding of:
- OWASP Top 10 web application security risks
- How to identify and exploit common web vulnerabilities
- Security testing and assessment techniques
- Defensive programming practices

## Lab Structure

Each lab includes:
- **Objectives**: What the lab covers
- **My Notes**: Detailed explanations and concepts
- **Practice**: Hands-on techniques I tried
- **Prevention**: How to defend against vulnerabilities
