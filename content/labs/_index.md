# IAW301 Lab Exercises
## Spring 2026 - Web Application Security

This page contains all 19 lab exercises for the IAW301 Web Application Security course. Each lab focuses on specific vulnerabilities and attack techniques commonly found in web applications.

## Course Learning Objectives

By completing these labs, students will:
- Understand the OWASP Top 10 web application security risks
- Learn to identify and exploit common web vulnerabilities
- Develop skills in security testing and assessment
- Understand defensive programming practices

## Lab Structure

Each lab includes:
- **Objectives**: What you will learn
- **Lab Content**: Detailed explanations and concepts
- **Practice Exercises**: Hands-on exploitation techniques
- **Prevention Measures**: How to defend against the vulnerabilities
