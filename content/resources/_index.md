# Learning Resources

## Recommended Learning Platforms

### PortSwigger Web Security Academy
- **URL**: https://portswigger.net/web-security/all-topics
- **Description**: Free, comprehensive web security learning platform
- **Why I recommend it**: Excellent hands-on labs with detailed explanations

### OWASP Resources
- **OWASP Top 10**: https://owasp.org/www-project-top-ten/
- **Web Security Testing Guide**: https://owasp.org/www-project-web-security-testing-guide/

## Security Tools

### Burp Suite
- **Download**: https://portswigger.net/burp/communitydownload
- **Type**: Web application security testing platform
- **Features**: Proxy, scanner, intruder, repeater

### SQLMap
- **Website**: https://sqlmap.org/
- **Type**: Automated SQL injection tool
- **Usage**: `python sqlmap.py -u "target_url"`

## Practice Environments

### DVWA (Damn Vulnerable Web Application)
- Intentionally vulnerable application for practice
- Great for beginners to learn exploitation techniques

### HackTheBox / TryHackMe
- Online platforms with various security challenges
- Good for continuous learning and skill development

## Useful Browser Extensions

- **Wappalyzer** - Identify technologies used by websites
- **Cookie Editor** - Manage and edit cookies
- **User-Agent Switcher** - Change browser user agent

## Books & Documentation

- OWASP Testing Guide
- Web Application Hacker's Handbook
- PortSwigger documentation

## Important Reminders

- **Always practice ethically** - Only test on authorized systems
- **Legal compliance** - Understand local laws regarding security testing
- **Use lab environments** - Don't test on production systems without permission
